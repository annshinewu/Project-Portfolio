# Project-Portfolio
There are three projects in this portfolio: TheWorldOfTomorrow, LibraryModel, and Minesweeper. 
The world of tomorrow is a business database created for the department of air traffic controlling the flying licenses, flying tests, employees, and etc. 
The library model is a four class java project with each class representing a book, bookshelf, user, and driverclass. 
The last project is an imitation of the game minesweeper utilizing 3 classes which are the individual tiles, the entire board, and the driver class.
